# Audio-Player-with-Equalizer
A MATLAB GUI-based audio player that allows setting an equalizer based on a different frequency. Basic audio players feature like play, pause, and stop are also included. The loudness can also be adjusted using a slider that is included. Along with the filtered waveform of the audio, it also displays the waveform of the original soundtrack.![Demo](https://user-images.githubusercontent.com/125946139/220253805-5b167dd2-3e9f-42de-95fc-7da06c6c6c6a.png)
